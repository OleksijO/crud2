This page represents Java web-application with simple CRUD functionality. Also this is rev.2 of mine Simple CRUD application.

In the top-right corner You can find site menu:
- Home: is page you looking at.
- Navigation: is page, where You can navigate through hierarchic category structure. Functions 'create', 'update' and 'delete' category or product are available after login as 'admin' ('admin1', ... ,'admin9') with password 'admin'.
- Database management: is page, where You can do some manipulations with database, of course, if you are logged as 'admin'.

Session timeout is set to 1 minute to show 'session expired behavior'. Just idle for 60 seconds and then click any button or link.

This application uses:
- Hibernate 4.2.2 for retrieve/edit/store category and product entities from/in database (PostreSQL 9.4);
- Spring 4.3.1 for inversion of control, database transaction management ;
- Mojarra 2.2.8 as Java Server Faces basic implementation;
- PrimeFaces 6.0 for adjust UI;
- SLF4J and Log4j for frameworks' and application logging.
