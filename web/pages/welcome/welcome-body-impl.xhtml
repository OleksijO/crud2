This page represent Java web-application with simple CRUD functionality. Also this is rev.2 of mine Simple CRUD application.

In the top-right corner You can find site menu:
- Home: is page you looking at.
- Navigation: is page, where You can navigate through hierarchic category structure. Functions 'create', 'update' and 'delete' category or product availible after login as 'admin' with password 'admin'.
- Database management: is page, where You can do some manipulations with database, of course, if you are logged as 'admin'.
